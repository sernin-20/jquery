= XMind Additional Tools For Windows =

== Clean.CMD ==

Deletes caches and persisted states generated by XMind application. You may
want to execute this command before uninstalling XMind to get your system clean.

WARNING: Use this script with caution!! It also deletes user-preferred things
like unsynced XMind Cloud file modifications, imported marker/clipart sets,
saved templates, cached license keys, keyboard shortcut definitions, etc.
